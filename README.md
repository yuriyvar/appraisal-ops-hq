# appraisal-ops-hq

Operations hub for our appraisal company: a self-improving knowledge system
(`vault/`), the Claude skills that run it (`skills/`), slash commands
(`.claude/commands/`), and a home for future MCP servers and scripts.

Start here:
1. `CLAUDE.md` — the rules of the road (Claude reads this automatically).
2. `vault/README.md` — how the memory system and its TPS loops work.
3. `BACKLOG.md` — what we're building, sprint by sprint.

Daily habit: append learnings to `vault/00-inbox.md` (or `/capture ...`).
Weekly: `/review`. Bi-weekly: `/retro`. SOPs change only via kaizen branches.
