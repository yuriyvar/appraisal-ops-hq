# Weekly review + kaizen retro (~20 min human time)

1. Run the memory-review skill -> approve/adjust its promotions and prunes.
2. Run the kaizen-retro skill -> it presents top kaizen items with 5-whys drafts
   and proposed SOP diffs.
3. Human decides: approve diffs (version bump), schedule PDCA check dates.
4. Update 50-metrics/metrics.md.
5. One question to close: "what made work harder this week that nobody logged?"
   -> append it to the inbox before leaving the meeting.
